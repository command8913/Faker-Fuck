# Faker-Fuck 🕵️

**Программа для обнаружения чита Faker**

---

## 📌 Что это?

**Faker-Fuck** — это инструмент для Windows, который определяет наличие софта, используемого для подмены серверов, перехвата трафика и создания фейковых сетевых окружений.

Программа проверяет систему по **11 признакам** и выдает результат: обнаружен Faker или нет.

---

## 🎯 Как это работает?

### Сетевые проверки
- Поиск характерных DNS-адресов (`mshome.net`) и IP-адресов (`192.168.137.1`)
- Проверка статуса mDNS (многоадресной DNS-рассылки)
- Обнаружение подмены Host-заголовков в HTTP-запросах
- Выявление прокси-соединений с аномальным поведением

### Системные проверки
- Поиск внедренных DLL в процессе `javaw.exe`
- Обнаружение виртуальных Wi-Fi адаптеров
- Проверка активности hostednetwork и мобильного хотспота

### Порт-мониторинг
- Анализ занятости порта 25565 (стандартный порт Minecraft-серверов)

### Сканирование сети
- Вывод всех устройств в локальной сети с их MAC-адресами, IP и именами
